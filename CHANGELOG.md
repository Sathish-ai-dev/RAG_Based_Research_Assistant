# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [Unreleased]

### Added
- Document upload feature via web UI
- Enter key support for sending queries
- Clean, simplified chat interface
- FastAPI web server with REST API endpoints
- Support for multiple LLM providers (OpenAI, Groq, Google Gemini)
- Vector database integration with ChromaDB
- Document chunking and embedding pipeline

### Changed
- Removed context/sources sidebar for cleaner UI
- Improved error handling and user feedback

### Added
- Initial project structure
- RAG assistant core functionality
- Document loading and processing
- Vector similarity search
- LLM integration with LangChain
- Web interface with FastAPI
